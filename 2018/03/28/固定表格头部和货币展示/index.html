固定表格头部和货币展示

背景:

今天遇到的两个问题：一， 固定后台报表的表格头部, 内容可以向下滚动.
二、给数字从右边每三位添加一个逗号(也就是货币展示)。

一， 固定后台报表的表格头部, 内容可以向下滚动.

一思路:

报表位置是不固定的,因此不能像导航栏一样直接写 position:fixed固定.
那应该怎么办呢?

方案一: 利用CSS3 + JS

给整个表格加 max-height: 500px; overflow: auto;
表格头部加transform: translateY(0px);
利用JS获得滚动的距离并且动态改变transform: translateY(0px)的值.
这样表格头部就是相对静止的,视觉上就是”固定的”
