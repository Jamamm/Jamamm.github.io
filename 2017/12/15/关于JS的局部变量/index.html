关于JS的局部变量的使用

1，不用全局属性作为id名，否则会相互覆盖。
2，立即调用函数：声明一个函数，然后立即调用。
3，声明一个函数，不要函数名，后面紧跟.call()，就可以仅仅使用它的局部变量。


function(){
    var  partent=………….
    ………….
}.call()

4，3这样做浏览器会报语法错误，这时候就需要用一点小技巧，比如
①用（）将它完整包起来


  (function(){
    var  partent=………….
    ………….
}.call())

②用（）只将函数体包起来


  (function(){
    var  partent=………….
    ………….
}).call()

③前面加一个 + 或 - 或 ! 或 ~


+function(){
    var  partent=1
    ………….
}.call()


-function(){
    var  partent=1
    ………….
}.call()


!function(){
    var  partent=1
    ………….
}.call()


~function(){
    var  partent=1
    ………….
}.call()

④JS很傻逼，连用个局部变量都这么麻瓜，迫切需要升级，于是let来了，用花括号{}包起来。（es6才有，2015年）


{
let  parent = 1
}

这样就没关系了！let不会变量提升，就不会覆盖原来的变量了。
